Overview

simple_json is a production-ready JSON library for Eiffel that builds on the standard eJSON library to provide modern features. It offers JSON Schema validation (Draft 7), JSON Pointer navigation (RFC 6901), JSON Patch operations (RFC 6902), JSON Merge Patch (RFC 7386), JSONPath queries, and comprehensive error tracking.

Part of the simple_* ecosystem of focused, single-purpose Eiffel libraries. Includes 29 HTML documentation files with EIS (F1 help) integration.

Quick Start

Installation

Add to your ECF file:

<library name="simple_json" location="$SIMPLE_JSON/simple_json.ecf"/>

Parse & Navigate

local
    value: SIMPLE_JSON_VALUE
    pointer: SIMPLE_JSON_POINTER
do
    value := json.parse ('{"users": [{"name": "Alice"}]}')
    create pointer.make_from_string ("/users/0/name")
    print (pointer.value_at (value))  -- "Alice"
end

Schema Validation

local
    validator: SIMPLE_JSON_SCHEMA_VALIDATOR
do
    create validator.make_from_string (schema_text)
    if validator.validate_string (data).is_valid then
        -- Valid JSON
    end
end

Features

  • JSON Schema Validation (Draft 7) - Only Eiffel library with this
  • JSON Pointer (RFC 6901) - Path-based navigation
  • JSON Patch (RFC 6902) - Standardized modifications
  • JSON Merge Patch (RFC 7386) - Declarative merging
  • JSONPath Queries - SQL-like queries for JSON
  • Streaming Parser - Process large files efficiently
  • Entity Serialization - SIMPLE_JSON_SERIALIZABLE pattern
  • 216 tests, 100% coverage
  • Full Design by Contract support
